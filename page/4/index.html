2019-06-25-如何将Windows10系统默认浏览器EDGE改为IE

如何将Windows10系统默认浏览器EDGE改为IE

WIN10 自带IE11。
点击开始——所有应用——windows附件——IE11。可以右键点击后选择“更多”——固定到任务栏或直接用鼠标拖到桌面，以方便使用。
如果附件里没有看到IE，就需要开启加载：
右键点击开始——程序和功能——启用或关闭windwos功能——勾选 IE11 ——确定，就会自动加载了。
另，将 IE11 设置为默认浏览器：
点击开始——设置——系统——默认应用——点击“web浏览器”下方的图标——选择默认程序为 IE 11 。

Windows10的附件在哪里

点击开始菜单－所有程序。
滚动到W字母开关的地方就可以看到Windows附件。
点击Windows附件就可以看到熟悉的画图、记事本等常用小工具。
右键单击画图，可以看到子菜单，点击“更多”子菜单。
点击弹出的“固定到任务栏”子菜单，可以将画图工具固定到任务栏。
现在就可以看到画图工具已经在任务栏了，其他小工具也可以这样找到并固定到任务栏或开始菜单。

注意：遇到要求装IE插件的网站，IE运行出错，则需要你以管理员权限来运行IE

windows10以管理员身份运行的设置方法
最简单方法：
直接右键要运行的exe程序文件，选择，以管理员身份运行，即可
另外其它方法
小编介绍windows10以管理员身份运行的设置方法，只需以下设置后，双击即可以管理员身份运行该程序了。
一、通过计算机管理永久开启Administrator管理员账号登录
右键桌面的“计算机” - 选择“管理”;依次展开“计算机管理(本地) - 系统工具 - 本地用户和组 - 用户”在右边的文件里面找到“Administrator”并双击它，在“常规”选项下将“账户已禁用”的勾去掉，这样就开启了 Administrator(管理员)账户。
windows10以管理员身份运行的设置方法
注：以后打开任何程序都会默认以管理员的身份运行。
二、对各别程序进行设置，让它以后以管理员身份运行。
右键你想要设置的程序然后选择“属性”;换到“兼容性”选项卡上，在特权等级栏下勾选“以管理员身份运行此程序”，单击“确定”。
windows10以管理员身份运行的设置方法(1)
三、在任意一个快捷方式上鼠标右键单击“属性”，切换到“快捷方式”选项卡，单击“高级”按钮，在“高级属性”对话框中，勾选以下“用管理员身份运行”的选项，注：快捷方式的管理员设置也可按照第二种方法。
windows10以管理员身份运行的设置方法(2)
以上就是windows10以管理员身份运行的设置方法介绍了，以上的方法设置后，打开程序系统就会默认从管理员身份运行了，很方便，

2019-06-05-Apple公司全球开发者大会WWDC(2019年6月)总结

WWDC

苹果全球开发者大会 [1] （英文全称是“Worldwide Developers Conference”），简称为“WWDC”，每年定期由苹果公司（Apple Inc.）在美国举办。大会主要的目的是让苹果公司向研发者们展示最新的软件和技术。

Apple公司全球开发者大会WWDC(2019年6月)总结

  • watchOS有了自己的应用商店
  • 在iOS 13里面，苹果创造了一个登陆保护机制“Sign in with Apple”，这其实是之前他们“强密码”功能的延展，以前只是苹果帮你生成密码，现在连登陆名都帮你随机生成了，彻底杜绝隐私滥用。
  • 苹果第五大系统诞生：iPadOS
  • Mac Pro：苹果最强电脑到来 Mac Pro是苹果公司电脑产品线里战斗力最高的产品，它定位是“工作站”，给专业人士用的专业电脑，比如电影特效制作公司等。
  • 这次的新Mac Pro是不锈钢框架+铝合金外壳，处理器当然是英特尔的至强Xeon，28核，服务器或工作站专用高性能CPU，一点也不意外；拓展能力也很彪：8个PCIe插槽，甚至网口都有俩。因为要支持至强处理器还有Radeon Pro Vega II这种耗电大户，这货的电源容量1500瓦，三个风扇散热。
  • 最强显示器“Pro Display XDR”
  • Mac Pro是个主机，自然需要一个与之配合的显示器。，32英寸LCD，6K分辨率级别。显示器日常亮度能到1000尼特，最高可达1600尼特，100万：1的对比度，6016x3384像素，共有1.2亿像素。实现真正的HDR显示（High Dynamic Range高动态范围）。他们觉得这个太厉害了，HDR这名字硬配不上它，所以干脆起了个新名字，显示器就叫“Pro Display XDR”，直译就是“专业显示器XDR”，简单粗暴。
  • macOS：iTunes退休 三个官方App取代
  • 新macOS最大变化就是把原来的iTunes拆成三个应用：Apple Music、Apple Podcasts和Apple TV。
  • 跟中国用户有关的新功能是Sidecar，可以让iPad变成Mac的第二块显示屏，唯有苹果全家桶才能最完美（就不让你买安卓平板……）。
  • 苹果还提升了语音控制功能，现在macOS和iOS可以实现完全语音操作，这是残障人士的福音。
  • 对码农来说，具有深远意义的改进是苹果的Project Catalyst功能，它能让帮助开发者将iOS应用变成Mac应用，这是给开发者节省时间的办法，也是弥补现在mac版App Store软件商店应用太少的尴尬。
  • 开发语言Swift建立的框架——SwiftUI，它的任务是让开发者门更快更简单的写出代码。
  • 而且它的界面很直观，左边代码，右边呈现效果。可以用于watchOS、tvOS、macOS等苹果旗下系统的App开发。
  • 这是码农们最开心的时刻，它拉低了写代码的门槛，让入门者能最快写出一个程序，在这个基础上开发的App，还能调用iOS设备上的现成框架，例如ARkit等。

2019-03-30-howto如何同步已fork的github工程到最新

2019-03-30-howto如何同步已fork的github工程到最新
1.浏览器中打开已fork的github工程 (打开你的github fork repo)
2.点击Pull request (在commit提示的那一行中) 或点击New pull request
(3.在这里，如果有出现，要点击switching the base的情况，就点)
4.点击Create pull request (默认情况下，github会比较original/your fork，这时应该不会有任何输出，因为你并没有做过任何变更)
5.弹个框出来，输入点什么，比如sync，然后点Create pull request
(6.在这里，如果有出现，就点击switching the base)
7.浏览器往下拉，点Merge pull request, 然后输入点什么或不输入，再点confirm
8.所有的改动都被你一网打尽了, 完成同步
